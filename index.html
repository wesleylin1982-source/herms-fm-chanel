🔧 系統設定
連接 Google Sheets 後台，讓報修資料自動寫入試算表並發送 Email 通知。

三步驟完成串接

1

建立 Google Sheets 試算表

前往 sheets.new 建立一個新的試算表，命名為「HERMS FM 報修系統」。

2

貼上 Apps Script 程式碼

在 Sheets 中點選「擴充功能」→「Apps Script」，將 HERMS_AppScript.gs 的內容全部貼上並儲存。
修改 FM_EMAIL 為您的 Email，然後點選「部署」→「新增部署作業」→ 類型選「網頁應用程式」→ 存取權選「所有人」→ 點「部署」，複製網址。

3

將 Apps Script 網址貼入下方

貼入後點「儲存連線」，系統即可自動將報修資料寫入您的 Google Sheets。

✅ 已連線至 Google Sheets

🚀 部署到網路（GitHub Pages）

A

申請免費 GitHub 帳號

前往 github.com 註冊（免費），建立一個新 Repository，命名為 herms-fm。

B

上傳 HTML 檔案

將 HERMS_FormFinal.html 重新命名為 index.html，上傳到 Repository。

C

啟用 GitHub Pages

進入 Repository → Settings → Pages → Source 選 main branch → Save。
約 1 分鐘後即可存取 https://你的帳號.github.io/herms-fm/
